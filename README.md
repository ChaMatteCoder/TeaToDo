# TeaToDo

TeaToDo is a calm, local-first productivity app built with React, TypeScript, Vite and Tailwind CSS.

## Features

- Daily task dashboard with quick and detailed task creation.
- Calendar, routines and planning notes.
- Focus timer with tea-inspired presets.
- Habit tracking with weekly progress.
- Smart lists for simple checklists, shopping and study plans.
- Theme customization, profile settings and local data backup/import.
- Local-first storage using the browser's `localStorage`.

## Getting Started

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run lint
npm run build
npm run qa
```

## Notes

The app currently runs fully on the client. User data is stored locally in the browser and can be exported or imported from the Personalizar page.
